# gh-analysis

Interactive visualizations for 5 popular GitHub repositories.

Built using:
Python + PyGitHub + Flask + Dash + Flask-Restful + Postgres + AWS 

Routes:
* Refresh DB (last 3 days): POST /api/refresh
* Visualization 1: GET /api/visualization1
* .....
